# Othello Modular Game

A fully modular Othello (Reversi) game implementation in Python.

## Features
 - Modular architecture (Logic / GUI / Players separated)
 - Automatic pass handling
 - AI and Human player support
 - Full unit testing suite
 - Complete documentation

 ## Installation
 1. Clone repository
 2. Create virtual environment: `python -m venv venv`
 3. Activate: `venv\Scripts\activate`
 4. Install: `pip install -r requirements.txt`

 ## Running Tests
 `pytest --cov=src`
